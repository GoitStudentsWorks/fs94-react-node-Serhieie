## Проєкт

Проєкт створений як застосунок ToDoList. Він забезпечує можливість додавання
різних зон для роботи, у кожній з яких можна додати необхідну кількість колонок.
Колонка є місцем для збору карток із завданнями за однією тематикою або на
одному етапі виконання. У картки із завданнями можна додати пріоритет та дату,
до якої необхідно виконати завдання.

Користувач має змогу створити декілька зон (дошок) для роботи та вільно
переключатися між ними. Це дозволяє організувати різні аспекти своєї діяльності
в одному застосунку, зберігаючи при цьому зручність та впорядкованість. Крім
того, є можливість призначення кольорових міток для кращої візуальної
організації завдань, що полегшує орієнтування в різних зонах та колонках.
Інтерфейс програми інтуїтивно зрозумілий та зручний у використанні, що робить
роботу з ним приємною та ефективною.

### Основні функції:

- **Додавання зон для роботи**: Користувач може створити різні зони (дошки) для
  різних проектів або аспектів діяльності.
- **Додавання колонок**: У кожній зоні можна створювати необхідну кількість
  колонок для організації завдань.
- **Картки із завданнями**: Кожна картка може містити завдання з пріоритетом і
  кінцевою датою виконання.
- **Переключення між зонами**: Легкий перехід між різними зонами для зручності
  роботи.
- **Кольорові мітки**: Можливість додавання кольорових міток для візуальної
  організації завдань.
- **Нагадування**: Функція нагадувань допомагає не забути про важливі завдання
  та терміни.

### Приємні бонуси:

- **Синхронізація теми**: Додаток знає, яку тему користувач обрав, і буде її
  відображати на всіх пристроях.
- **Приємні фони**: Застосунок пропонує приємні фони та можливість їх зміни, що
  забезпечує комфортний робочий процес.
- **Кросплатформенність**: Підтримка роботи на різних платформах дозволяє
  використовувати додаток на будь-якому пристрої з збереженням налаштувань.

### Інтерфейс

Інтерфейс програми розроблений таким чином, щоб бути інтуїтивно зрозумілим і
зручним для користувача. Це дозволяє легко створювати та керувати завданнями, що
робить роботу з ToDoList приємною та ефективною.

### Використані технології

1. HTML/SCSS: Відповідає за структуру та стилізацію інтерфейсу.
2. JavaScript: Додає інтерактивності та логіки до застосунку.
3. React (та бібліотеки): Дозволяє створювати динамічні компоненти
   користувацького інтерфейсу.
4. Axios: Використовується для запитів до серверу.
5. Node.js: Забезпечує роботу серверної частини додатка.
6. Vite: Використовується для швидкої розробки та збірки проєкту.
